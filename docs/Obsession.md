## Privilege Escalation
For a more comfortable shell, we fix the tty:

```
script /dev/null -c bash
stty raw -echo; fg
reset xterm
export TERM=xterm
```

We check sudo permissions and exploit this to spawn a root shell:

```
russoski@c1489df97f06:~$ sudo -l
```
